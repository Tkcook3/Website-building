What is smart parking

Smart parking is when drivers can find parking spots easier without causing traffic congestion,fuel wastage,frustation and pollution.

About the Project

This project uses smart sensors to track parking availability in real-time. By connecting hardware to the web, drivers can see exactly where spots are open before they even arrive.
