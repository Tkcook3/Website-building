Our Components used

Our system mainly uses these 6 core components:

  • 1. Ultrasonic Sensors: Detects if a car is parking in the spot.
  • 2. Microcontroller: The component that processes sensor data.
  • 3. Wi-Fi Module: Sends the data to the cloud.
  • 4. LED Indicators and Screen: Shows red or green lights for drivers for open parking spots or displays at entry points.
  • 5. Power Supply: Lets the system run 24/7.
  • 6. Cloud Database: Stores real-time spot availability.
